- Postgres: >=9.5

* Clone this repo
- `https://github.com/Laguna1/c1linic-manager`
- `cd C1linic-manager` 

* System dependencies
- `bundle install`
* Configuration

* Database creation
- `rails db:create`

* Database initialization
- `rails db:migrate`

* Start server
- `rails s`
- Open `http://localhost:3000/` in your browser


* How to run the test suite
- `rspec`
